### 6.1. TinyString API Coverage

The `tinystring` package provides equivalents for most common operations:

```go
// String manipulation
Convert("hello").ToUpper().String()           // "HELLO"
Convert("HELLO").ToLower().String()           // "hello"
Convert("hello world").Replace(" ", "-")      // "hello-world"

// HTML/Attribute escaping
Convert("<script>").EscapeHTML()              // "&lt;script&gt;"
Convert("value\"test").EscapeAttr()           // Safe for HTML attributes

// String building
b := Convert()
b.Write("Hello")
b.Write(" ")
b.Write("World")
html := b.String()                            // "Hello World"

// Formatting (instead of fmt.Sprintf)
Fmt("Hello %s, you are %d years old", name, age)

// Path operations (instead of path.Join)
PathJoin("output", "index.html").String()     // "output/index.html"
```

### 12.6. Utility Functions (from tinystring)

```go
// String conversion and manipulation
Convert(s string) *Conv                    // Create converter
(*Conv).String() string                    // Get final string
(*Conv).ToUpper() *Conv                   // Convert to uppercase
(*Conv).ToLower() *Conv                   // Convert to lowercase
(*Conv).Replace(old, new string) *Conv    // Replace substring
(*Conv).EscapeHTML() string               // Escape HTML entities
(*Conv).EscapeAttr() string               // Escape for HTML attributes

// String building
Convert() *Conv                            // Create empty builder
(*Conv).Write(s string)                   // Append string
(*Conv).WriteInt(i int)                   // Append integer
(*Conv).Reset()                           // Clear buffer

// Formatting
Fmt(format string, args ...any) string    // Printf-style formatting

// Path operations
PathJoin(parts ...string) *Conv           // Join path segments
```
